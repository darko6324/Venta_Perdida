¿Cómo solucionar?
  • Opción 1 (más fácil): Abre una terminal en la carpeta donde está el HTML y ejecuta:
    python -m http.server 8080
    Luego abre http://localhost:8080/quick_bi_dashboard.html en tu navegador.
  • Opción 2: Usa la extensión Live Server de VS Code y abre el archivo desde ahí.
  • Opción 3 (Chrome): Cierra Chrome, ábrelo con: chrome --allow-file-access-from-files y luego abre el archivo.
  • Verifica que tengas conexión a internet y que el sheet de Google esté publicado como público.
